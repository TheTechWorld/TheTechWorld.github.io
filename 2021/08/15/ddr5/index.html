DDR5 RAM将比DDR4快得多，但有一个重大进步

Corsair是准备很快发布DDR5内存的品牌之一。DDR5的运行温度可能比DDR4高得多，因为它们已经将电压调节从主板本身移开，现在它位于[RAM模块]上，因此您实际上可能会产生更多的热量。

DDR5 内存即将面世，领先的制造商正在为它的发布做准备。Corsair 是准备很快发布 DDR5 RAM 的品牌之一。该公司透露了更多关于 DDR5 RAM 的可能性，以及一些困难，包括它比 DDR4 运行得更热这一事实。

海盗船内存 Corsair DDR5 RAM

热身并不意外。在最好的 RAM 在 DDR5 阵容将是高达 4 倍更强大的比它的前身，因此产生更多的热量。不过，据海盗船 DIY 营销总监 George Makris 表示，原因还在于内存模块本身的设计。

“可以想象，DDR5 的运行温度可能比 DDR4 高得多，因为它们已经将电压调节从主板本身移开，现在它位于 [RAM 模块] 上，因此您实际上可能会产生更多的热量，”George Makris 在 YouTube 上说 Corsair 频道上发布的视频。

已经宣布新的 DDR5 内存将具有以前闻所未闻的速度。即使是最低时钟速度也将等于或超过 DDR4 RAM 生成的最高数字。Corsair 表示数据传输速率将从 6,400 MT/s 开始，最高可达 12,600 MT/s。预计该技术将支持高达 8,400MHz 的时钟。

除了速度提高之外，每个 RAM 棒都将具有更大的容量。虽然 DDR4 RAM 的每个双列直插内存模块 (DIMM) 的最大容量为 32GB，但新的 DDR5 内存将能够支持高达 128GB。如此令人印象深刻的升级必然需要更好的冷却。Corsair 表示，只有通过启用板载电源管理集成电路 (PMIC) 和电压调节模块 (VRM) 才能实现这种升压。

散热器从来都不是 RAM 制造商真正考虑的重点——这适用于 DDR3 和 DDR4 时代。当涉及到 DDR5 时，这可能仍然会改变，因为记忆棒变得更加复杂和耗电的硬件。Corsair 声称它在制造新的 DDR5 RAM 时使用的当前冷却系统将继续保持温度稳定。

DDR5 技术的开发考虑到了未来的硬件。对于今天的标准，性能的提升似乎有点矫枉过正，但几年后它可能会成为常态——而 DDR5 旨在支持这种进步。

通过多种设计选择，可以实现面向未来的数据传输速率扩展。DDR5 RAM 模块带有自己的 PMIC 和 VRM，允许进行一些严重的超频——但这些组件需要额外的冷却。这得到了电压增加的支持。Adata 已经在为其顶级 DIMM 测试 1.6 伏设计。标准是 1.1 伏，这意味着比我们目前使用的电压增加了 45%。这种过电压会产生更多的热量。

海盗船计划在今年晚些时候发布其首款 DDR5 内存模块，可能会在截止日期前与英特尔第 12 代处理器一起发布。冷却这些功能强大的内存条是否会成为一个问题仍有待观察，但这是海盗船提前意识到这一点的好兆头。
