Windows 11 Insider Previews：最新版本中的新特性？

在向 Windows 预览体验成员推出时，获取有关 Windows 11 新预览版本的最新信息。现在针对 2021 年 8 月 12 日发布的 Insider Preview Build 22000.132 进行了更新。

6 月下旬，微软宣布即将发布 Windows 11，现在正在开发和测试频道中向 Windows Insiders 发布早期预览版本。不过，并非所有人都能参与 Windows 11 Insider 计划，因为新操作系统的系统要求比 Windows 10更严格。

如这篇 Microsoft 博客文章所述，如果您的 PC 无法满足 Windows 11 的最低硬件要求，您将无法加入 Insider Dev 或 Beta 频道。您可以改为加入 Release Preview Channel 以测试 Windows 10 预览更新。

如果您在 6 月 24 日之前是 Windows 10 Insider Dev Channel 的成员并且您的 PC 不满足硬件要求，您仍然可以收到 Windows 11 内部版本。但微软警告称，“由于这些设备不符合新的硬件要求，可能存在影响 Windows 11 在这些 PC 上的体验的问题和错误，这些问题可能无法修复。” 如果出现需要您恢复到 Windows 10 的问题，您将无法接收更多的 Windows 11 版本。

在 Windows 11 正式发布后，任何不满足最低要求的 PC 都不会收到 Windows 11 Insider Preview 版本。

下面，我们将提供每个 Windows 11 预览版本中的内容摘要，从最新版本开始。对于每个版本，我们都包含了发布日期、发布到哪个 Insider 频道、版本中的内容摘要以及指向 Microsoft 相关公告的链接。

Windows 11 Insider Preview 版本
Windows 11 Insider Preview Build 22000.132
发布日期： 2021 年 8 月 12 日

通过此版本，Beta 频道中的预览体验成员现在可以从 Microsoft Teams 访问聊天。该功能之前在 Dev Channel 中向 Insiders 推出，包括一对一和群组音频和视频通话。您可以创建和加入会议、打开或关闭麦克风和摄像头，以及选择您喜欢的扬声器、麦克风和摄像头。您还可以共享您的屏幕、查看参与者名单、从大厅接纳会议参与者、聊天以及在图库视图中查看人们的视频。有关更多详细信息，请参阅“ Microsoft Teams 聊天的第一个预览开始向 Windows 预览体验成员推出”。

该版本还推出了适用于 Windows 11 的新截图工具，并为开发者频道中的 Insiders 更新了计算器、邮件和日历应用程序。有关详细信息，请参阅“面向 Windows 11 的 Windows 预览体验成员推出的第一组 Windows 应用更新”。

此版本中有七个修复程序，包括任务栏在 Windows 沙盒内反复崩溃的错误，以及另一个“使用中的位置”指示器图标未在任务栏中显示的错误。

此版本中有许多已知问题，其中包括在使用从开始或任务栏搜索时您可能无法输入文本的问题，以及在单击任务栏上的搜索图标时搜索面板可能无法打开的问题。

Windows 11 Insider Preview Build 22000.120
发布日期： 2021 年 8 月 5 日

发布到： Dev 和 Beta 频道

此版本为 MSA 帐户（单一登录 Microsoft 帐户）引入了一个新的家庭小部件。它可以让人们查看其 Microsoft 家庭组成员的近期活动。该版本还修复了几十个错误，其中一个错误是文件资源管理器在黑暗模式下没有显示更新的滚动条；另一个导致“设置”中的搜索处于永久损坏状态，每次搜索都说“无结果”；还有一种是在单击“任务视图”按钮时 explorer.exe 崩溃。

此版本中有许多已知问题，其中包括在使用“从开始”或任务栏搜索时您可能无法输入文本的问题，以及在单击任务栏上的“搜索”图标后可能无法打开“搜索”面板的另一个问题。

（获取有关Windows 11 Insider Preview Build 22000.120 的更多信息。）

Windows 11 Insider Preview Build 22000.100
发布日期： 2021 年 7 月 22 日

发布到：开发频道

此版本将 Microsoft Teams 集成引入 Windows 11，包括任务栏上的 Teams Chat 弹出窗口。有关更多详细信息，请参阅 Microsoft 的博客文章“ Microsoft Teams 聊天的第一个预览开始向 Windows Insiders 推出”。请注意，并非每个 Insider 都会立即看到该功能。

还有各种小新功能，包括在单击顶角的 V 形标记时使任务栏日历弹出窗口完全折叠，以便为通知留出更多空间。

修复了几十个错误，包括一个当您右键单击桌面或文件资源管理器时，生成的上下文菜单和子菜单部分出现在屏幕外，另一个在某些情况下无法登录小部件到认证挂。

此版本中存在许多已知问题，其中包括在使用“从开始搜索”或任务栏时您可能无法输入文本的问题，以及在切换输入法时任务栏有时会闪烁的另一个问题。

Windows 11 Insider Preview Build 22000.71
发布日期： 2021 年 7 月 15 日

发布到：开发频道

此版本引入了一个新的娱乐小部件，可在 Microsoft Store 中显示新的和精选的电影标题。还有许多错误修复，包括在“文件资源管理器选项”>“查看”下启用“在单独的进程中打开文件夹”时新命令栏没有出现的问题，以及另一个使设置在启动时定期崩溃的问题。

此版本中存在许多已知问题，其中包括从“开始”或任务栏使用搜索时您可能无法输入文本的问题，以及桌面或文件资源管理器中的右键单击菜单和子菜单可能会部分关闭的问题-屏幕。

Windows 11 Insider Preview Build 22000.65
发布日期： 2021 年 7 月 8 日

发布到：开发频道

在此版本的其他更改中，“开始”菜单现在包括一个搜索框、“设置”中的“电源和电池”页面上现在提供电源模式设置，以及多个系统警报对话框，例如电池电量不足时的警报或当您更改显示设置时，现在坚持新的 Windows 11 视觉设计。

该版本包括各种错误修复，其中包括无法启动设置、无法从开始固定和取消固定应用程序、文件资源管理器中的命令栏消失的另一个问题，以及另一个无法启动直到您重新启动 PC，布局才会出现。

此外，Windows Print Spooler 服务中的远程代码执行漏洞（称为“PrintNightmare”）已被关闭。有关详细信息，请参阅CVE-2021-34527。

此版本中有许多已知问题，其中包括在使用从开始或任务栏搜索时您可能无法输入文本的问题，以及另一个在“在单独的进程中打开文件夹”时可能不会出现新命令栏的问题在File Explorer Options > View下启用。

IT 需要了解的内容：因为这是一个安全更新，所以应该相对较快地应用。在接下来的几周内，检查有关有问题的问题的报告，如果一切正常，请应用更新。

Windows 11 Insider Preview Build 22000.51
发布日期： 2021 年 6 月 28 日

发布到：开发频道

第一个适用于 Windows 11 的 Insider Preview 是一个很大的预览版，它提供了许多预计将成为新操作系统发行版的一部分的新功能。

一般界面更改： “开始”菜单现在居中，而不是固定在屏幕左侧。它包括一组固定的应用程序以及 OneDrive 中所有连接云的设备（包括 PC 和智能手机）上最近打开的文件。任务栏也居中并提供新的动画。它在右下角有一个按钮，用于通知和快速更改设置。

文件资源管理器具有新的命令栏和新的整体外观，包括新的上下文菜单。新主题已添加到整体界面，Windows 声音已重新设计。

小工具： Windows 11 附带一组可调整大小的信息小工具，用于处理新闻、天气、日历、本地交通、待办事项列表、OneDrive 中的照片、新闻、体育等内容。

多任务处理： Windows 10 的“Snap”功能让您可以控制如何安排打开的窗口，从而得到改头换面。您可以从预先构建的 Snap 布局中进行选择，并将它们排列到预先设计的 Snap 组中。通过任务视图可用的桌面现在可以重新排序并具有自定义背景。

触摸、墨迹书写和语音输入： 触摸键盘获得 13 个新主题并且可以调整大小。还有一个语音输入启动器和几个新的触摸手势。该版本可让您通过出现在任务栏右下方快速设置旁边的切换器在其他语言和键盘之间快速切换。现在还有一个用于自定义数字笔的菜单。

其他变化包括：

Microsoft Store已重新设计。最终，Android 手机游戏和应用程序将安装在那里，并且能够在 Windows 11 上运行。
当您卸下笔记本电脑时，外接显示器上的窗口将最小化。当您将计算机重新连接到显示器时，Windows 会将所有内容放回原来的位置。
借助 Windows 11 的动态刷新率，您的 PC 可以在您输入墨迹或滚动时自动提高刷新率，为您提供更流畅的体验，并在您不需要时降低刷新率以节省笔记本电脑的电量。
设置已经过重新设计，包括在页面之间持续存在的左侧导航。还有面包屑导航可帮助您准确了解自己在“设置”中的位置。
Windows 11 提供Wi-Fi 6E。
已知问题：此版本中有超过两打已知问题，涉​​及各种功能，包括任务栏、设置、开始、搜索、小部件和应用商店。例如，将具有多个用户帐户的设备升级到 Windows 11 时，设置将无法启动；单击“搜索”图标任务栏后，“搜索”面板可能无法打开；当从 Windows 10 升级到 Windows 11 或安装到 Windows 11 的更新时，某些功能可能会被弃用或删除。
