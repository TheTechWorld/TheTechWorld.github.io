AIdea 大语言模型和会话的开源app重磅发布

发表于2023-09-07|更新于2024-09-08|编程
|阅读量:

AI大模型可真是越来越成熟，各种集成应用层出不穷！
AIdea 是一款集成了主流大语言模型以及绘图模型的 APP， 采用 Flutter 开发，代码完全开源

支持多种大预言模型，支持图片生成式！

主要功能

  1. 支持 GPT-3.5/4 问答聊天
  2. 支持国产模型：通义千问，文心一言
  3. 支持文生图、图生图、超分辨率、黑白图片上色等功能，集成 Stable Diffusion 模型，支持 SDXL 1.0

开源代码地址

客户端：https://github.com/mylxsw/aidea
服务端：https://github.com/mylxsw/aidea-server （即将开放）

app下载地址

Github Release 地址： https://github.com/mylxsw/aidea/releases/tag/1.0.4

app更新内容

本次更新内容如下

解决 Mac 端应用图标过大问题，现在图标大小跟大部分 App 保持一致，圆角风格也做了统一
Bugfix: 首页 “历史纪录” 应该为 “历史记录”
Bugfix: Web 端聊天界面 input 影响输入法的第一个字符
替换首屏加载中图标：增加加载中请稍候提示
Mac 端、Web 端现在已支持支付宝充值
Web 端 wasm 文件和 Google 字体文件修改问从本地加载，不再从 Google 下载
