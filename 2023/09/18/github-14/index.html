让windows用户可以体验airpods的强大的免费工具

AirPodsDesktop 是一个开源免费 AirPods 桌面用户体验增强程序，适用于Windows 系统，作者GitHub
曾经有个付费软件叫 MagicPods，据称它可以实现「为 Windows 提供魔法般的 AirPods 体验」，AirPodsDesktop 为使用 AirPods 耳机和 Windows 系统的同学提供了另外一个选择
